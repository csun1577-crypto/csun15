        <h3>👍 좋은 점 (사업성이 괜찮은 이유)</h3>
        <ul>

          <li><b>초기 비용이 비교적 낮음</b>: 대형 창업 대비 부담이 덜하고 1인 운영 가능</li>
          <li><b>단가가 나쁘지 않음</b>: 가정용 1대 8~15만 원 수준, 시스템/업소용은 더 높음</li>
          <li><b>기술 진입장벽 ‘중간’</b>: 교육+실습으로 숙련 가능, 전문가 이미지 만들기 유리</li>
        </ul>

        </div>

          <li><b>체력 소모</b>: 분해·세척·조립이 반복되어 허리/어깨 부담</li>

        <h2>2) 피지컬 로봇 시대가 오면? (미래 변화 방향)</h2>

        <h3>단기: “바로 대체” 가능성은 낮음</h3>
        <ul>
          <li>가정/매장 환경은 비정형(모델·설치·오염 패턴이 제각각)</li>
          <li>로봇이 어려워하는 영역: 예외 처리, 다양한 구조 분해/재조립</li>
        </ul>

        <h3>중·장기: 단순 작업은 로봇화, 사람은 ‘결정+책임’으로 이동</h3>
        <ul>
          <li><b>로봇화 가능성이 큰 영역</b>: 필터/기본 외부 청소, 규격화된 대형 시설(빌딩) 유지보수 일부</li>
          <li><b>사람의 가치가 큰 영역</b>: 진단(악취·곰팡이 원인), 예외 상황 대응, 고객 설명·신뢰, 책임</li>
        </ul>

        </div>

        <ul>

          <li>로봇/장비/사람을 조합해 생산성 높이는 <b>운영자</b></li>
          <li>고객 신뢰 기반 <b>지역 서비스업</b>으로 고정 고객 확보</li>
